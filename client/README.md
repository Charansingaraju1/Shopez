# ShopEZ

ShopEZ is a Stock Portfolio Management Web Application built using React, Node.js, Express, and MongoDB.

## Features

- User Registration
- User Login
- Buy Indian Stocks
- Buy US Stocks
- Portfolio Management
- Sell Stocks
- Profile Page
- Logout System
- MongoDB Database

## Tech Stack

### Frontend
- React.js
- React Router DOM

### Backend
- Node.js
- Express.js

### Database
- MongoDB

## Installation

### Client

```bash
cd client
npm install
npm run dev
```

### Server

```bash
cd server
npm install
node index.js
```
